This Dashboard widget will keep track of downloads from wordpress.org of specific plugins.

== Description ==
My Plugin Stats, will show the download stats of your custom plugins. It pull the information from wordpress.org. It creates a dashboard widget and you can monitor many plugins at once.
You can add and remove as many plugins as you like.


Some of the features:
* Add Plugins you want to watch stats on.
* Ful intergrated dashboard widget.
* Compatible with Wordpress (have not tested on BuddyPress or Mu)
* Will work with any caching system.

== Installation ==
1. Upload this plugin to the `/wp-content/plugins/` directory and unzip it, or
simply upload the zip file within your wordpress installation.

2. Activate the plugin through the 'Plugins' menu in WordPress

3. If you want to add other plugins go to the "Settings -> My Plugin Stats" and
add or remove the plugins of your choice.
 3a. example wordpress link for stats "http://wordpress.org/extend/plugins/wordpress-woot-watcher/stats/"

4. Add the dash board widget.
